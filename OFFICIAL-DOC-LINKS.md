## 補足

- 現行の出題範囲の正本は Microsoft Learn の GH-300 学習ガイドです。
- GitHub Docs 側は、学習ガイドの各項目を深掘りするための一次ドキュメント集として使うのが安全です。
- Learn の skill group 名と GitHub Docs のカテゴリ名は完全には一致しません。特に `Use GitHub Copilot features` と `GitHub Copilot features` は Learn 側で重複して見えるため、実際には関連ドキュメントをまとめて追う必要があります。
- 日本語版が存在しない、または同等の個別ページを確認できなかったものだけ英語 URL を残しています。
