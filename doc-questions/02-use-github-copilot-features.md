## 問1

Copilot Free の IDE 内チャットについて正しいものはどれですか。

- A) 50 メッセージ / 月まで利用できる
- B) 200 メッセージ / 月まで利用できる
- C) 利用回数は無制限である
- D) IDE 内チャットは利用できない

**解答: A**

**解説:** Copilot Free では、IDE 内でのチャット利用が 50 メッセージ / 月までと案内されています。

## 問2

GitHub Enterprise Server では、現時点で利用できないと明記されているものはどれですか。

- A) GitHub Copilot Free
- B) GitHub Copilot
- C) GitHub Copilot Chat
- D) GitHub Copilot CLI

**解答: B**

**解説:** プラン比較では、GitHub Copilot は GitHub Enterprise Server では現在利用できないとされています。

## 問3

IDE で Copilot Chat を使うための条件として、正しいものはどれですか。

- A) GitHub Advanced Security の有効化
- B) GitHub Mobile のインストール
- C) Copilot へのアクセス権と対応 IDE
- D) Enterprise Managed Users の利用

**解答: C**

**解説:** IDE のチャット機能は、Copilot へのアクセスと、対応している IDE が前提です。

## 問4

Copilot CLI の認証で、GitHub CLI がインストール済みで認証されている場合に使われるものはどれですか。

- A) GitHub Actions の実行トークン
- B) リポジトリの deploy key
- C) SSO のみ
- D) GitHub CLI のトークン

**解答: D**

**解説:** Copilot CLI は GitHub CLI のトークンをフォールバックとして利用できます。

## 問5

プラン比較の「Other features」に含まれるものはどれですか。

- A) GitHub Codespaces
- B) GitHub Pages
- C) GitHub Spark
- D) GitHub Issues のテンプレート

**解答: C**

**解説:** GitHub Spark は、プラン比較の「Other features」に含まれています。

## 問6

プラン比較の「Customization」に含まれる項目を 2 つ選んでください。

- A) Audit logs
- B) Repository and personal custom instructions
- C) Prompt files
- D) GitHub Spark

**解答: BC**

**解説:** カスタマイズの対象には、リポジトリおよび個人向けのカスタム命令と、prompt files が含まれます。

## 問7

プラン比較の「Agents」に含まれる項目を 2 つ選んでください。

- A) Copilot coding agent
- B) GitHub Spark
- C) Audit logs
- D) Agent mode

**解答: AD**

**解説:** Agents には Copilot coding agent と Agent mode が含まれます。

## 問8

プラン比較の「Copilot Chat」に並ぶ利用先のうち、IDEs を除いた 3 つを選んでください。

- A) GitHub Pages
- B) GitHub Mobile
- C) GitHub
- D) Windows Terminal

**解答: BCD**

**解説:** Copilot Chat の利用先として、GitHub Mobile、GitHub、Windows Terminal が案内されています。

## 問9

Copilot CLI の関連ドキュメントで扱われている内容を 3 つ選んでください。

- A) 対話的に使うことやセッションの扱い
- B) スクリプトの生成とファイル操作
- C) 認証の設定
- D) GitHub Spark の作成

**解答: ABC**

**解説:** Copilot CLI のドキュメントでは、対話利用、セッション、スクリプト生成、ファイル操作、認証設定が扱われています。

## 問10

プラン比較の「Other features」に含まれるものを 3 つ選んでください。

- A) Copilot pull request summaries
- B) Audit logs
- C) Agent mode
- D) Content exclusion

**解答: ABD**

**解説:** Other features には、Pull Request summaries、Audit logs、Content exclusion が含まれます。
